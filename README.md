## 1、项目介绍

在线考试管理系统拥有两种角色

管理员：用户管理、角色管理、试题管理、试卷管理等

学生：开始和查看错题集


## 2、项目技术

后端框架： Servlet、mvc模式

前端技术：jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA
- tomcat版本：Tomcat 7-9
- 数据库类型：MySql（5.x和8.x版本都可） 
- maven项目：否
- 硬件环境：Windows 或者 Mac OS
